# 부록 B. 이주민 사용자 인터뷰 질문지

## 인터뷰 원칙
"좋은 아이디어인가요?"를 묻지 말고, **최근 아팠던 실제 경험과 그때 한 행동**을 묻는다. 개인 건강정보는 말하지 않아도 된다고 안내하고 **익명**으로 정리한다.

## 시작 멘트
"안녕하세요. 저는 한국에 사는 외국인이 아플 때 건강정보와 병원 이용을 더 쉽게 이해하도록 돕는 방법을 조사하고 있습니다. 오늘은 제품 판매나 의료 상담이 아니라, 실제 경험을 듣고 싶습니다. 정답은 없습니다."

## 핵심 질문 (경험 기반)
1. 어느 나라에서 오셨고, 한국에 오신 지 얼마나 되었나요?
2. 한국어로 병원·약국에서 대화하는 건 어느 정도 편하신가요?
3. 한국에서 아팠던 적이 있나요? 가장 최근 경험을 들려주세요.
4. 그때 **가장 먼저 무엇을** 했나요? (검색 / 가족·지인 / 번역앱 / 약국 / 병원 / 그냥 참기)
5. 왜 그 방법을 선택했나요?
6. 그 과정에서 **가장 어려웠던 점**은 무엇이었나요?
7. 건강정보를 찾을 때 주로 어디를 쓰나요? 그 정보를 믿을 만하다고 느꼈나요? 왜요?
8. 병원에 가야 할지 말지 판단이 어려웠던 적이 있나요?
9. 병원에서 증상을 설명하기 어려웠던 적이 있나요?
10. 건강보험·진료비·예약·진료과 선택에서 어려움이 있었나요?

## MVP(부록 A)를 보여준 뒤
- 이 안내문이 무엇을 하는 서비스라고 이해하셨나요?
- 가장 도움이 된 부분 / 가장 이해하기 어려운 부분은?
- 실제로 열·기침이 날 때 사용할 것 같나요?
- 이해 쉬움 / 신뢰 / 사용 의향을 각각 1~5점으로.
- ChatGPT나 검색보다 나은 점이 있나요? 부족한 점은?
- 어떤 언어로 제공되면 좋겠나요?

## 신뢰 조건 질문
- 이 안내를 믿으려면 무엇이 필요할까요? (공공기관 출처 / 의사 감수 / 기관(센터·보건소) 연결 / 지인 추천 / 후기)
- "의료 전문가 감수 전 초안"이라고 쓰여 있어도 사용하시겠어요?
- 다문화센터·보건소가 이 링크를 보내주면 더 믿을 수 있나요?
- 무료로 제공되면 사용할 것 같나요?

## 성공 기준 (5명 인터뷰 시)
- 3명 이상이 병원·약국 이용에 어려움을 겪었다고 말함
- 3명 이상이 기존 정보(검색·번역앱)를 완전히 신뢰하지 못함
- 3명 이상이 병원에서 증상 설명이 어려웠다고 말함
- MVP 이해도·신뢰도 평균 4점 이상 `[가정]`
- 3명 이상이 실제 사용 의향 있음
