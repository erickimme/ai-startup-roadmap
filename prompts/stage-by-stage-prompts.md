# 단계별 복사용 프롬프트

## Stage 1. 아이디어 정의서

```text
내 아이디어를 아이디어 정의서로 정제해줘. 출력은 엘리베이터 피치, 타겟 고객, 문제, 기존 대안, 차별점, 가장 약한 가정, 다음 검증 질문 순서로 작성해줘.
```

## Stage 2. BMC

```text
지금까지의 아이디어 정의서를 바탕으로 9블록 BMC를 작성해줘. 각 블록마다 가장 중요한 가정을 하나씩 표시하고, 숫자가 필요한 곳은 [가정]으로 표시해줘.
```

## Stage 3. GTM

```text
초기 고객을 어디서 어떻게 데려올지 GTM 전략을 작성해줘. P0/P1/P2 세그먼트, 채널, 첫 10명 획득 방법, 12주 런칭 플랜, 핵심 메시지를 포함해줘.
```

## Stage 4. 세일즈 플랜

```text
수익 모델, 가격, 퍼널, 첫 12개월 매출 가정을 작성해줘. CAC, 전환율, ARPU는 근거가 없으면 [가정]으로 표시하고 민감도도 함께 보여줘.
```

## Stage 5. 개발 로드맵

```text
MVP 개발 로드맵을 작성해줘. 반드시 만들 기능, 만들지 않을 기능, 기술 스택, 12주 개발 마일스톤, 개인정보/보안 고려사항을 포함해줘.
```

## Stage 6. HR 플랜

```text
초기 팀과 채용 계획을 작성해줘. 지금 필요한 역할, 3/6/12개월 채용 순서, 정규직/계약직/외주 판단, 채용 트리거, 월 인건비 가정을 포함해줘.
```

## Stage 7. 실행 타임라인

```text
3개월, 6개월, 12개월 실행 타임라인과 분기별 OKR을 작성해줘. 가장 큰 리스크 5개와 완화 전략도 포함해줘.
```

## Stage 8. 피어 리뷰

```text
VC와 회의적인 동료 창업자 관점에서 내 산출물을 공격적으로 리뷰해줘. 강점, 갭, 치명적 질문, 수정 우선순위를 작성해줘.
```

## Stage 9. 최종 수정본과 피치

```text
피어 리뷰를 반영해서 최종 변경 요약과 30초/60초/3분 피치를 작성해줘. 예상 질문 5개와 답변도 포함해줘.
```
