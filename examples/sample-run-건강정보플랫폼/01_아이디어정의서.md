# 01. 아이디어 정의서 — 건강이음(HealthBridge)

> 코치 메모: 초기 아이디어 "의료 정보 접근성이 낮은 사람들"은 범위가 너무 넓어 질문으로 P0를 좁혔습니다.
> - 누가 가장 심각? → **국내 거주 비영어권 외국인·이주민**(언어 + 한국 의료제도 이중 장벽)
> - 지금 무엇으로? → 가족·지인 통역, 부정확한 번역앱, 커뮤니티 입소문
> - 왜 부족? → 의료 용어 오역, 진료과·건강보험 제도 모름, 오정보 위험
> - 왜 지금? → 체류 외국인 250만 시대, LLM 다국어 품질 급상승
> - 왜 창업자? → 의료기기산업·국제보건 배경, 의료윤리 관점

## 정제된 엘리베이터 피치
**한국에 사는 비영어권 외국인·이주민**이 아프거나 가족이 아플 때 **어느 병원·진료과에 가고, 증상·약을 어떻게 이해할지 막막할 때**, **건강이음**이 **신뢰할 수 있는 공공·의료기관 출처 기반의 모국어 건강정보 안내**로 **'어디로 가서 무엇을 준비할지'를 명확하게** 해 준다.

## 핵심 가치 제안 5블록
| 블록 | 내용 |
|---|---|
| 누구를 위해 | 국내 거주 비영어권 외국인·이주민(다문화 가정, 유학생, 이주노동자) |
| 어떤 문제를 | 언어+제도 장벽으로 '어느 병원·진료과, 증상·약 이해, 비용'이 막막 |
| 어떻게 해결 | 신뢰 출처 기반 모국어 안내 + 진료과/병원 연결 + 방문 준비 체크리스트 |
| 왜 지금 | 체류 외국인 250만+ , LLM 다국어 의료 안내 품질 상승 |
| 왜 우리 | 의료기기산업·국제보건 배경, 의료윤리(진단 아닌 정보) 원칙 |

## 기존 대안과 차별점
| 기존 대안 | 쓰는 이유 | 부족한 점 | 우리의 차별점 |
|---|---|---|---|
| 번역앱(파파고 등) | 즉시·무료 | 의료 용어 오역, 제도 모름 | 의료 특화 + 출처 신뢰도 표시 |
| 가족·지인 통역 | 신뢰·맥락 | 항상 가능X, 전문성 한계 | 24시간·일관 품질 |
| 다문화센터 상담 | 사람 응대 | 대기·시간 제약 | 즉시·반복·확장 |

## 가장 약한 가정
- 사용자가 **AI 건강정보**를 (오정보 우려에도) **신뢰**하고 의사결정에 활용할 것 `[가정]`
- 모국어 의료 안내의 **정확도**가 신뢰를 줄 만큼 충분할 것 `[가정]`
