# 부록 A. 발열·기침 MVP 안내문 1장 (초안)

**제목: 한국에서 열이 나고 기침이 날 때 / When You Have Fever and Cough in Korea**
> ⚠️ 이 안내문은 **진단·처방이 아닙니다.** 병원 방문 전 준비를 돕는 정보입니다. 실제 서비스는 **베트남어·중국어**부터 우선 제공합니다.

## 1. 먼저 확인하세요: 위험 신호 (이 중 하나라도 있으면 즉시 병원/119)
| 한국어 | English |
|---|---|
| 숨쉬기 어렵다 | Difficulty breathing |
| 가슴 통증이 있다 | Chest pain |
| 의식이 흐리거나 심하게 어지럽다 | Confusion or severe dizziness |
| 고열이 계속된다 | Persistent high fever |
| 증상이 빠르게 심해진다 | Symptoms worsening quickly |
| 물도 못 마실 만큼 힘들다 | Trouble drinking fluids / dehydration |

## 2. 쉬운 설명
열·기침·목 통증·콧물·몸살은 감기, 독감, 코로나19 같은 호흡기 감염에서 나타날 수 있습니다. **증상만으로 병명을 확정할 수 없습니다.** 이 안내문은 병명을 맞히는 것이 아니라 병원 방문 준비를 돕습니다.

## 3. 어디로 가야 할까요?
| 상황 | 고려할 곳 |
|---|---|
| 증상이 가볍고 기본 상담 | 보건소 / 학교·기관 보건실 |
| 일반 감기약 상담 | 약국 |
| 열이 계속·증상이 심함 | 병원/의원 |
| 숨쉬기 어렵거나 급격히 악화 | 응급실 또는 **119** |
| 건강보험·진료비 문의 | 국민건강보험공단 / 다문화센터 |

## 4. 병원에서 말할 문장
| English | 쉬운 한국어 |
|---|---|
| I have had a fever since yesterday. | 어제부터 열이 있어요. |
| I have a cough and sore throat. | 기침이 나고 목이 아파요. |
| My symptoms started ___ days ago. | 증상은 ___일 전부터 시작됐어요. |
| I am allergic to ___. | 저는 ___ 알레르기가 있어요. |
| My Korean is not good. Please explain slowly. | 한국어가 서툴러요. 천천히 설명해 주세요. |

## 5. 의사에게 물어볼 질문
| English | 쉬운 한국어 |
|---|---|
| Do I need any tests? | 검사가 필요한가요? |
| Do I need medicine? | 약을 먹어야 하나요? |
| When should I come back? | 언제 다시 와야 하나요? |
| What symptoms mean I should go to the ER? | 어떤 증상이면 응급실에 가야 하나요? |

## 6. 피드백 질문 (테스트용)
- 이 안내문이 이해하기 쉬웠나요? (1~5)
- 신뢰할 수 있다고 느꼈나요? (1~5)
- 실제로 아플 때 사용할 것 같나요? (예/아니오)
- 가장 도움이 된 부분 / 가장 헷갈린 부분은?
- 어떤 언어가 더 필요한가요?
