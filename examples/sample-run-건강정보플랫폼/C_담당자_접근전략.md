# 부록 C. 다문화센터·지자체·보건소 담당자 접근 전략

## 접근 원칙
처음부터 **팔지 않는다.** 이주민 인터뷰 증거를 들고 "기관이 이미 겪는 문제인지" 배우러 간다. 구매는 문제 합의 이후다.

## 처음에 묻지 말아야 할 질문
- 이 서비스 사시겠어요?
- 예산 있으세요?
- 파일럿 하실래요?
- 연 ○○○만원이면 가능하세요?

## 처음에 물어야 할 질문 (문제 조사)
1. 외국인 주민이 아플 때 센터/보건소로 문의하는 경우가 있나요?
2. 주로 어떤 문의가 **반복**되나요?
3. 지금은 어떤 안내 자료·절차를 쓰시나요?
4. 언어 문제나 병원 안내에서 어려운 점이 있나요?
5. 외부 건강정보 안내 도구를 주민에게 공유하려면 어떤 조건이 필요한가요? (출처·감수·개인정보·승인 절차)

## 첫 이메일 예시
> **제목:** 외국인 주민의 병원 이용 어려움 관련 짧은 인터뷰 요청드립니다
>
> 안녕하세요. 저는 의료기기산업·국제보건을 공부하는 ○○○입니다. 한국에 사는 비영어권 이주민이 아플 때 건강정보와 병원 이용을 얼마나 쉽게 이해하는지 조사하고 있습니다. **제품 판매가 아니라**, 주민들이 실제로 어떤 어려움을 겪는지, 센터에서는 어떻게 안내하시는지 배우고 싶습니다. 가능하시면 **20분 정도** 짧게 인터뷰를 요청드립니다.

## 좋은 신호 / 나쁜 신호
| 확인할 것 | 좋은 신호 | 나쁜 신호 |
|---|---|---|
| 문의 빈도 | 자주 들어온다 | 거의 없다 |
| 반복 업무 | 매번 같은 설명을 한다 | 이미 자료가 충분하다 |
| 리스크 인식 | 잘못 안내할까 조심스럽다 | 우리 업무 범위가 아니다 |
| 배포 채널 | 오리엔테이션·문자·홈페이지 가능 | 외부 서비스 공유 어렵다 |
| 의사결정 | 센터·보건소장 선에서 검토 가능 | 본부·법무·개인정보 심사가 복잡 |

> 목표: "사주세요"가 아니라 **"같이 파일럿으로 검증해 봅시다"**까지 가는 것. 좋은 신호가 3개 이상이면 무료 파일럿을 제안한다.
