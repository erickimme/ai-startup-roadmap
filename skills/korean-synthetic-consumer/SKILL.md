# Korean Synthetic Consumer

## 목적

한국 고객 페르소나 3-7명을 구성해 제품/메시지/가격/반대 논리를 빠르게 점검한다.

## 안전 문구

합성 패널은 가설 생성 도구입니다. 시장 검증 완료, 수요 증명, 투자 판단 근거로 쓰면 안 됩니다. 반드시 실제 인터뷰와 행동 실험으로 이어가야 합니다.

## 모드

- `persona-panel`: 빠른 다중 페르소나 첫 반응
- `product-fit`: 사용 순간, 대안, 도입 트리거, WTP
- `message-test`: 랜딩 카피, 피치, 아웃리치 문구 테스트
- `objection-map`: 구매/도입 반대 논리와 필요한 증거
- `interview-script`: 실제 인터뷰 질문 생성
- `full-report`: 전체 리포트

## 입력

- 제품/아이디어 설명
- 타겟 세그먼트
- 원하는 행동: 클릭, 가입, 결제, 상담, 추천 등
- 가격/패키징 가정
- 테스트하고 싶은 문구

## 패널 구성 규칙

최소 5명 권장:

1. 얼리어답터
2. 메인스트림 사용자
3. 예산/결재권자
4. 회의적인 사용자
5. 엣지 케이스 또는 비사용자

## 출력 공통 형식

```markdown
## 합성 소비자 패널 결과

### 패널 구성
| # | 페르소나 | 세그먼트 | 상황 | 편향/주의 |
|---|---|---|---|---|

### 반응 분포
- 긍정:
- 조건부:
- 부정:

### 핵심 인용
> 페르소나:

### 주요 반대 논리
| 반대 | 심각도 | 필요한 증거 | 피해야 할 반박 |
|---|---:|---|---|

### 추천 수정
- 유지:
- 바꿀 것:
- 제거할 것:

### 다음 실제 검증
- 인터뷰 대상:
- 질문:
- 행동 실험:
```

## 안티패턴

- 긍정 반응만 뽑지 않는다.
- WTP를 확정 가격으로 해석하지 않는다.
- "한국인 전체" 같은 넓은 세그먼트로 패널을 만들지 않는다.
