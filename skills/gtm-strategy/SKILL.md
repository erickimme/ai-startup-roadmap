# GTM Strategy

## 역할

그로스 마케터처럼 본다. "광고하면 된다"가 아니라 첫 10명, 첫 100명을 어디서 어떻게 데려올지 쓴다.

## 입력

- 아이디어 정의서
- BMC
- 타겟 고객
- 창업자의 기존 네트워크
- 사용 가능한 예산과 시간

```markdown
## GTM 전략

### 타겟 세그먼트 우선순위
| 우선순위 | 세그먼트 | 구매/사용 트리거 | 도달 경로 | 왜 먼저인가 |
|---|---|---|---|---|
| P0 |  |  |  |  |
| P1 |  |  |  |  |
| P2 |  |  |  |  |

### 첫 10명 획득 계획
1.
2.
3.

### 채널 전략
| 단계 | 목표 | 채널 | CAC 가정 | 핵심 KPI |
|---|---|---|---:|---|
| 시드 |  |  |  |  |
| 성장 |  |  |  |  |
| 확장 |  |  |  |  |

### 12주 런칭 플랜
| 주차 | 목표 | 활동 | 검증 지표 |
|---|---|---|---|

### 핵심 메시징
- 헤드라인:
- 서브 헤드라인:
- 증거:
- CTA:

### 가장 약한 가정
```
