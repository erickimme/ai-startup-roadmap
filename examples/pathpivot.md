# PathPivot — 천장(Ceiling) 샘플 산출물

> 이 문서는 **상위 10% 수준의 완성 예시**입니다. 참가자는 70% 수준만 나와도 성공입니다.
> 구조만 빌리고 내용은 본인 아이디어에 맞추세요. (사용법: `examples-pathpivot-note.md`)
>
> 아이디어 한 문장: **"3~7년차 직장인이 자신의 경험을 살려 현실적인 커리어 전환 경로를 찾도록 돕는 AI 커리어 피벗 코치."**
> 모든 숫자는 예시이며 근거 상태를 `[가정]`/`[근거]`로 표시했습니다.

---

## 1. 아이디어 정의서

### 정제된 엘리베이터 피치
**이직을 고민하지만 무엇으로 옮길지 모르는 3~7년차 직장인**이 막연함에 멈춰 있을 때, **PathPivot**이 **본인 경력·강점 데이터를 분석해 3개의 현실적 전환 경로와 90일 실행 플랜**을 제시해 **"다음 한 걸음"을 명확하게** 만든다.

### 핵심 가치 제안 5블록
| 블록 | 내용 |
|---|---|
| 누구를 위해 | 전환을 고민하는 3~7년차 직장인(특히 비개발 직군) |
| 어떤 문제를 | "옮기고 싶지만 어디로, 어떻게"가 막연함 |
| 어떻게 해결 | 경력 입력 → AI가 전환 경로 3개 + 90일 플랜 + 필요한 역량/포트폴리오 |
| 왜 지금 | AI로 1:1 커리어 코칭 수준을 저비용으로 자동화 가능 |
| 왜 우리 | 그로스/HR 도메인 경험 + 한국 채용 데이터 해석력 |

### 기존 대안과 차별점
| 기존 대안 | 쓰는 이유 | 부족한 점 | 우리의 차별점 |
|---|---|---|---|
| 커리어 유튜브/블로그 | 무료, 접근 쉬움 | 내 상황에 안 맞음 | 내 경력 기반 개인화 |
| 헤드헌터 | 실제 자리 연결 | 전환(피벗)엔 무관심 | 피벗 경로 설계가 핵심 |
| 오프라인 커리어 코칭 | 깊이 있음 | 회당 10만원+, 느림 | 즉시·저비용·반복 가능 |

### 가장 약한 가정
- 직장인이 커리어 조언에 **돈을 낼** 만큼 절박한가? `[가정]`

### 다음 검증 질문
1. 최근 6개월 내 이직·전환을 검색·상담한 적 있는가?
2. 그때 무엇에 돈/시간을 썼는가?
3. AI 코치 결과를 신뢰하려면 무엇이 필요한가?

## 3. GTM 전략

### 타겟 세그먼트 우선순위
| 우선 | 세그먼트 | 트리거 | 도달 경로 | 왜 먼저 |
|---|---|---|---|---|
| P0 | 3~5년차 마케터 | 연봉협상/평가 시즌 | 링크드인·마케팅 뉴스레터 | 전환 검색 활발, 결제력 |
| P1 | 영업/CS 직군 | 번아웃 | 직장 익명 커뮤니티 | 규모 큼 |
| P2 | 비개발 기획자 | 조직개편 | 커뮤니티/추천 | 입소문 |

### 첫 10명 획득 계획
1. 본인 링크드인에 "전환 경로 무료 진단 10명" 공개 모집
2. 마케팅 단톡방 3곳에 사례 1건 공유
3. 진단 받은 사람에게 후기 + 1명 추천 요청

### 채널 전략
| 단계 | 목표 | 채널 | CAC `[가정]` | KPI |
|---|---|---|---:|---|
| 시드 | 첫 50명 | 핸드 아웃리치 | ₩0 | 진단 완료율 |
| 성장 | 월 500명 | 뉴스레터 협찬·콘텐츠 | ₩12,000 | 가입→결제 |
| 확장 | 월 3,000명 | 퍼포먼스 광고 | ₩18,000 | LTV/CAC |

### 핵심 메시징
- 헤드라인: "이직 말고, **피벗**. 당신 경력으로 갈 수 있는 3가지 길."
- 증거: 실제 전환 사례 3건
- CTA: "무료로 내 전환 경로 보기"

### 가장 약한 가정
- 핸드 아웃리치가 광고보다 싸게 첫 고객을 줄 것 `[가정]`

## 5. 개발 로드맵

### MVP 범위
- **포함**: 경력 입력 폼 → AI 경로 3개 생성 → 90일 플랜 → PDF. 결제(1회).
- **제외**: 추적 대시보드, 커뮤니티, 추천 채용 연동.

### 핵심 사용자 흐름
1. 경력 3~5문항 입력 → 2. 경로 3개 + 플랜 확인 → 3. 결제 후 PDF 저장

### 기술 스택
| 레이어 | 선택 | 이유 |
|---|---|---|
| 프론트 | Next.js | 빠른 랜딩+폼 |
| 백엔드 | Supabase | 인증·DB 빠름 |
| 결제 | 토스페이먼츠 | 국내 |
| AI | GPT/Claude API | 경로 생성 |

### 12주 마일스톤(요약)
| 주차 | 마일스톤 | 검증 기준 |
|---|---|---|
| 1-2 | 랜딩+사전결제 | 결제 의향 |
| 3-6 | 경로 생성 v1 | 진단 완료율 |
| 7-12 | 결제+PDF | 유료 전환 |

### 가장 약한 가정
- AI 경로 품질이 "돈 낼 만큼" 신뢰를 줄 것 `[가정]`

## 9. 최종 1페이지 요약 + 미니 피치

### 최종 1페이지
- 고객: 3~7년차 비개발 직장인 / 문제: 전환 막연함 / 해결: AI 경로 3개+90일 플랜
- 차별점: 경력 기반 개인화 / 수익: 1회+구독 / MVP: 12주 / 팀: 1+계약 / 다음 검증: 사전결제

### 60초 피치
"이직은 검색되는데, **피벗**은 검색이 안 됩니다. 3~7년차 직장인 절반이 '옮기고 싶지만 어디로'에서 멈춰요. PathPivot은 당신 경력을 넣으면 갈 수 있는 **3가지 현실적 길과 90일 플랜**을 줍니다. 첫 진단은 커피 한 잔 값. 지난 2주 핸드 모집으로 받은 진단 50건 중 후기 90%가 '다음 행동이 명확해졌다'였습니다. 지금 필요한 건 사전결제 100건으로 수요를 증명하는 것, 함께 보실 분을 찾습니다."

### 남은 가장 큰 리스크
무료→유료 전환율. 이번 주 사전결제 테스트로 검증.
