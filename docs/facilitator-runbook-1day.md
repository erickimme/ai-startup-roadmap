## 세팅 블록 (12:10–12:45) — 다 함께, 체크포인트로

각 단계마다 **"되면 손"** 으로 진도를 맞추고, 안 된 사람은 조교가 1:1 투입.

1. **터미널 열기** → `node -v` 실행. 버전이 나오면 ✋
   - 안 나오면: https://nodejs.org LTS 설치(조교 지원). Windows는 WSL 권장.
2. **Codex 설치**: `npm install -g @openai/codex` → `codex --version` 확인 ✋
   - 권한 오류: macOS `sudo npm i -g ...` / Windows 관리자 PowerShell. (자세히: `docs/troubleshooting.md`)
3. **로그인**: `codex` 실행 → **워크숍용 ChatGPT 계정**으로 로그인 ✋
4. **시작**: README §3 프롬프트 붙여넣기 → 본인 아이디어 입력 → `git clone`·파일쓰기 **승인(y)** → 코치가 "Stage 1 시작"하면 ✋

> ⛑ **폴백 규칙**: 12:45까지 설치가 안 된 참가자는 **웹 ChatGPT + `prompts/bootcamp-agent-prompt.md`** 복붙 방식으로 수업을 따라가고, 휴식 시간에 설치를 마저 한다. **설치 때문에 전체 진도를 멈추지 않는다.**
