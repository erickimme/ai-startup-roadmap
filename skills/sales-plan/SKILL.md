# Sales Plan

## 역할

Pricing Lead와 세일즈 리더처럼 본다. 고객이 실제로 돈을 내는 구조인지 확인한다.

## 입력

- BMC
- GTM 전략
- 고객 세그먼트
- 기존 대안 가격
- 제공 가치

## 출력 형식

```markdown
## 세일즈 플랜

### 수익 모델
| 상품 | 가격 | 포함 내용 | 예상 매출 믹스 | 리스크 |
|---|---:|---|---:|---|

### 가격 앵커링
| 비교 대상 | 가격 | 우리 가격과의 관계 | 시사점 |
|---|---:|---|---|

### 퍼널 모델
| 단계 | M3 가정 | M6 가정 | M12 가정 | 근거 상태 |
|---|---:|---:|---:|---|
| 방문/노출 |  |  |  | [가정] |
| 리드 |  |  |  |  |
| 상담/체험 |  |  |  |  |
| 결제 |  |  |  |  |
| 유지/재구매 |  |  |  |  |

### 12개월 매출 목표
| 월 | 신규 고객 | 누적 고객 | 월 매출 | 주요 이벤트 |
|---|---:|---:|---:|---|

### 핵심 KPI
- 재무:
- 획득:
- 전환:
- 유지:

### 가장 약한 가정
```

## 보조 스킬

- 지불의사와 반대 논리는 `korean-synthetic-consumer`의 `product-fit`, `objection-map`으로 점검한다.
- 경쟁 가격 근거는 `evidence-research`로 찾는다.
