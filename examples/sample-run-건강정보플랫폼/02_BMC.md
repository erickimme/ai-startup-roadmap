# 02. 비즈니스 모델 캔버스 — 건강이음

| 블록 | 내용 | 가장 중요한 가정 |
|---|---|---|
| 고객 세그먼트 | 비영어권 이주민(B2C) · 다문화센터·지자체·병원 국제진료센터(B2G/B2B) | 누가 '돈 내는 고객'인가 |
| 가치 제안 | 신뢰 출처 기반 모국어 건강정보 + 진료과 안내 + 방문 준비 | 정확도·신뢰 |
| 채널 | 다문화센터·이주민 커뮤니티·교회/모스크·지자체 제휴 | 채널 도달률 |
| 고객 관계 | 셀프서브 챗 + 위급 시 유선/기관 연계 | 재사용 빈도 |
| 수익 흐름 | B2G/B2B 라이선스(지자체·병원) + 후원/그랜트, B2C는 무료 | 기관 지불의향 |
| 핵심 자원 | 검증된 의료정보 DB·출처, 의료 자문, 다국어 모델 | 의료 감수 확보 |
| 핵심 활동 | 콘텐츠 검증, 모델 안전성, 기관 제휴 | 오정보 차단 |
| 핵심 파트너 | 보건소·다문화센터·병원 국제진료, 공공 의료데이터 | 제휴 성사 |
| 비용 구조 | LLM 호출, 의료 감수 인건비, 제휴/운영 | 감수 비용 규모 |

> 핵심 통찰: 사용자(이주민)와 지불자(기관)가 다른 **B2G/B2B 모델**일 가능성이 높다. → 세일즈 플랜에서 검증.

## TAM / SAM / SOM (bottom-up)
| 레이어 | 정의 | 숫자 | 근거 | 신뢰도 |
|---|---|---:|---|---|
| TAM | 국내 체류 외국인 | 약 250만 명 | 출입국 통계 | 중간 |
| SAM | 비영어권·의료접근 취약 | 약 120만 명 | 국적 분포 추정 `[가정]` | 낮음 |
| SOM Y1 | 제휴 지자체·센터 이용자 | 약 1~3만 명 | 제휴 5곳×이용률 `[가정]` | 낮음 |
